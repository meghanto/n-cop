How to play: Click an edge to claim/block it for the current player. Cop can click up to n edges on their turn (or press “End Cop turn”). Robber clicks exactly one edge.
Cop view: Switch to deletions layout to hide blue edges and recompute a force-directed layout from the same ring start each time. Use robber-attraction to additionally pull red-connected vertices closer.
Tip: Hide neutral edges for k≥10. Hold Shift and drag to pan. Use mouse wheel or +/- to zoom.
